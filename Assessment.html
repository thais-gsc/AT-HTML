Design responsivo

Quando é recomendando a criação de um site separado para mobile? Seria este o caso deste site da GK Fashion?

Resposta:

É recomendado criar um site separado para mobile caso já exista um site pronto, pois seriam utilizados menos códigos, apenas para a versão mobile, e assim o site rodaria mais rápido. Não é o caso do GK Fashion, que como foi feito do zero, é preferível criar um site responsivo, podendo analisar como este vai funcionar em diferentes telas, através do uso de media queries no CSS.

Multimídia na Web com HTML5 e CSS3

O elemento audio permite como elemento‐filho o elemento source. Quais são os atributos que o source permite e qual conteúdo eles suportam? Quais os outros atributos do elemento audio e quando devem ser utilizados?

Resposta:

Source permite os atributos src e type, e suporta os conteúdos MP3, WAV E OGG.
O elemento audio possui os atributos src, para especificar a url do áudio; autoplay, para começar a tocar imediatamente; controls, para mostrar os controles do áudio como tocar e pausar; loop, para tocar repetidamente; muted, para especificar que o áudio deve ficar mudo; e preload, que especifica se o áudio deve pré-carregar junto com o carregamento da página.

O elemento video permite como elemento‐filho o elemento source. Quais são os atributos que o source permite e qual conteúdo eles suportam? Quais os outros atributos do elemento video e quando devem ser utilizados?

Resposta:

Source permite os atributos src e type, e suporta os conteúdos MP3, WebM E OGG.
O elemento video possui os atributos src, para especificar a url do vídeo; autoplay, para começar a rodar imediatamente; controls, para mostrar os controles do vídeo como ligar e pausar; loop, para rodar repetidamente; muted, para especificar que o áudio do vídeo deve ficar mudo; preload, que especifica se o vídeo deve pré-carregar junto com o carregamento da página; height e width, para estabelecer a altura e largura do vídeo; e poster, para escolher uma imagem que será apresentada enquanto o vídeo carrega ou até o usuário apertar o botão de início.

Resposta:

É necessário o uso de APIs. APIs (interfaces de programação de aplicações) são um conjunto de definições, protocolos e ferramentas para desenvolvimento de um software, sem que seja necessário envolver-se em detalhes nele, apenas utilizar seus serviços, no caso das APIs do HTML5, estas definições e protocolos são no próprio HTML.
